## 🚀 Projects

### [TrainFlow](https://github.com/apoorav21/TrainFlow)
AI-powered fitness coaching app with a fully serverless AWS backend. Ingests 20+ HealthKit
biometrics daily into DynamoDB via 24 Lambda functions, and runs an agentic GPT-5 coaching
loop with function calling for real-time plan adjustments and health summaries. Deployed as
a single CDK stack. Native Apple Watch companion included.

`Python` `AWS Lambda` `DynamoDB` `API Gateway` `AWS CDK` `OpenAI API` `SwiftUI`

---

### [API Testing AI Agent](https://github.com/apoorav21/API-Testing-AI-Agent)
Paste an OpenAPI spec, get a full suite of positive, negative, and boundary test cases
generated and executed automatically by an AI agent. Outputs a detailed, downloadable
test report.

`Python` `OpenAI API` `REST APIs`

---

### [AI Duel](https://github.com/apoorav21/AI-Duel)
Head-to-head evaluation framework for running two AI models against the same prompt or
task and comparing their outputs side by side. Built to make model evaluation faster
and more systematic.

`TypeScript` `LLM Evaluation`

---

### [LLM Knowledge Base](https://github.com/apoorav21/llm-knowledge-base)
Self-compiling knowledge base — drop articles in, get a structured wiki out. Inspired
by Karpathy's approach to personal knowledge management with LLMs.

`Shell` `LLMs`
